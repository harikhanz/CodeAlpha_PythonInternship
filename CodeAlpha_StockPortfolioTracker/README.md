# 📈 Stock Portfolio Tracker

## Description
A Python program that calculates the total value of a stock portfolio using predefined stock prices.

## Features
- Add stock symbols
- Enter stock quantities
- Calculate total investment value
- Simple and user-friendly interface

## Technologies Used
- Python
- Dictionaries
- Loops
- User Input

## How to Run

bash
python stock_tracker.py
