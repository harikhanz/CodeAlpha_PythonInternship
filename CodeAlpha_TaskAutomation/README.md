# 📂 File Organizer (Task Automation)

## Description
A Python script that automatically organizes files into folders based on their file extensions.

## Features
- Organizes images
- Organizes videos
- Organizes documents
- Organizes Python files
- Organizes archives

## Technologies Used
- Python
- os Module
- shutil Module

## How to Run

bash
python file_organizer.py
